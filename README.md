# 🏦 Análisis de Préstamos Bancarios con Python

## 📌 Descripción

Proyecto de análisis de datos enfocado en la evaluación y aprobación de préstamos bancarios utilizando Python.

El objetivo fue identificar los principales factores que influyen en la aprobación o rechazo de solicitudes de préstamo mediante análisis exploratorio de datos (EDA), cálculo de KPIs y visualizaciones.

---

## 🎯 Objetivos del Proyecto

- Analizar solicitudes de préstamos.
- Identificar variables relevantes para la aprobación.
- Evaluar el impacto del historial crediticio.
- Analizar ingresos y capacidad de pago.
- Construir visualizaciones para la toma de decisiones.

---

## 🛠️ Herramientas Utilizadas

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 KPIs Principales

| KPI | Valor |
|------|---------|
| Total de Solicitudes | 491 |
| Préstamos Aprobados | 343 |
| Préstamos Rechazados | 148 |
| Tasa de Aprobación | 69.86% |

---

## 📈 Análisis Realizados

### 1. Estado del Préstamo

Se analizó la distribución entre préstamos aprobados y rechazados.

### 2. Historial Crediticio

Se identificó como el factor más importante para la aprobación de préstamos.

### 3. Ingresos del Solicitante

Se evaluó la relación entre ingresos y aprobación.

### 4. Estado Civil

Se analizó la influencia del estado civil sobre la aprobación.

### 5. Educación

Se compararon graduados y no graduados.

### 6. Dependientes

Se evaluó el impacto de la cantidad de dependientes sobre la aprobación.

### 7. Monto del Préstamo

Se analizó la relación entre el monto solicitado y la aprobación.

---

## 🔍 Principales Hallazgos

### Historial Crediticio

El historial crediticio fue la variable con mayor influencia sobre la aprobación de préstamos.

### Ingresos

Los ingresos contribuyen a medir la capacidad de pago, pero no garantizan la aprobación.

### Dependientes

Los clientes con menos dependientes presentaron una mayor cantidad de préstamos aprobados.

### Estado Civil y Educación

Ambas variables mostraron cierta influencia, aunque menor que el historial crediticio.
